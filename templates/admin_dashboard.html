🔤 Jumble Words Manager

Add content here and the system will automatically generate jumble word puzzles for players

⚠️ Database Setup Required

Run this SQL in your Supabase SQL Editor to create the required tables:

CREATE TABLE IF NOT EXISTS jumble_contents (
    id           BIGSERIAL PRIMARY KEY,
    content_text TEXT        NOT NULL,
    added_by     VARCHAR(42) NOT NULL,
    word_count   INTEGER     DEFAULT 0,
    created_at   TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS jumble_words (
    id          BIGSERIAL PRIMARY KEY,
    content_id  BIGINT      NOT NULL REFERENCES jumble_contents(id) ON DELETE CASCADE,
    word        VARCHAR(100) NOT NULL,
    jumbled     VARCHAR(100) NOT NULL,
    created_at  TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_jumble_words_content ON jumble_words (content_id);
CREATE INDEX IF NOT EXISTS idx_jumble_contents_created ON jumble_contents (created_at);

ℹ️ How It Works

Paste any content paragraph (about GoodDollar, crypto, UBI, etc.) and the system will automatically:

  • Extract meaningful words (5+ letters) from your content
  • Create scrambled versions of each word
  • Add them to the player game pool automatically

Example: Content: "GoodDollar is a universal basic income" → Jumble word: "ruvinesla" (universal), players must type "universal" to win 5 G$.

➕ Add New Content

Only words with 5+ letters will be used. Common words (the, that, with...) are automatically filtered out.
